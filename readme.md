# Configuration des fichiers dockerfile et compose.yaml

Les fichiers `dockerfile` ou `compose.yaml` sont placés chancun dans des repertoires spécifiques et précis selon leur utilié.

- autodeclic-python : dossier contenant l'ensemble des fichiers nécessaire au déploiement de l'application python, notament les fichiers `Dockerfile` et `Dockerfile.preprod` ansi que `compose.prod`
- autodeclic-react : dossier conte
